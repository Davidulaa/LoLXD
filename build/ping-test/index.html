There is no way to see ping before entering a game (or practise tool etc). To correct for this, not only would League's game servers need to be pinged, they would need to be pinged frequently. Presumably this would pose an issue for Riot otherwise it is a mystery why it hasn't already been implemented, so to prevent unintentional spam, it hasn't been included in this tool either.

The reason frequent pings would be necessary, rather than just sending a single ping, is because network consistency makes a far greater impact on in-game performance than raw ping. If you are skeptical of this, please feel free to test for yourself on the simulation.
